# MiniMind 第7集《理论：RMSNorm》

## 层归一化基础原理
- 定义：将一层神经元的输出进行标准化
- 目的：稳定训练、加速收敛
- 核心思想：让每层的输出分布保持稳定

### 关键统计量
- 均值 μ
- 方差 σ
- 归一化公式：x̂ = (x - μ) / √(σ² + ε)

## LayerNorm vs RMSNorm 对比

### LayerNorm（层归一化）
- 计算所有特征的均值和方差
- 公式包含均值归一化：x̂ = (x - μ) / √(σ² + ε)
- 优点：理论完善，效果稳定
- 缺点：计算均值增加开销

### RMSNorm（均方根归一化）
- 只计算均方根（RMS），不计算均值
- 公式简化：x̂ = x / RMS(x)
- 优点：计算更快，效果相当
- 核心思想：均值不影响模型性能

### 对比结论
- 两者最终效果相近
- RMSNorm 去掉均值计算是合理的
- 实践中常替代 LayerNorm 使用

## RMSNorm 数学推导

### 符号定义
- x：输入向量
- n：特征维度
- ε：极小常数（防止除零）

### 推导过程
- 计算均方根：RMS(x) = √[(1/n) × Σxᵢ²]
- 归一化输出：x̂ = x / RMS(x)
- 可学习缩放：y = x̂ × g

### 核心洞察
- 去掉均值平移：分布形状不变
- 仅保留缩放信息：足够用于学习
- 计算量减少：O(n) → O(n) 但常数更小

## 代码实现要点
- 只需计算 L2 范数
- 不需要计算均值
- 添加可学习参数 g

## 应用场景
- Transformer 各层
- 语言模型
- 替代原始 LayerNorm
