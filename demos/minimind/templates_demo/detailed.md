## 1. 层归一化概述

### 1.1 什么是归一化？

归一化（Normalization）是深度学习中的核心技巧之一，其主要作用包括：

| 作用 | 描述 |
|------|------|
| **稳定梯度** | 缓解梯度消失/爆炸问题 |
| **加速收敛** | 使损失曲面更平滑 |
| **提升泛化** | 一定程度上防止过拟合 |
| **打破对称性** | 避免神经元学习到相同的特征 |

### 1.2 归一化的基本思想

归一化的核心公式形式为：

$$
\hat{x} = \frac{x - \mu}{\sigma} \cdot \gamma + \beta
$$

其中：
- $x$：输入数据
- $\mu$：均值
- $\sigma$：标准差
- $\gamma, \beta$：可学习的缩放和偏移参数

### 1.3 归一化方法的分类

根据计算均值和标准差的维度不同，归一化方法可分为：

```
归一化方法
├── BatchNorm    - 沿 batch 维度
├── LayerNorm    - 沿 feature 维度
├── RMSNorm      - 沿 feature 维度（不含均值）
├── GroupNorm    - 按 channel 分组
└── InstanceNorm - 单通道归一化
```

## 2. LayerNorm 详解

### 2.1 LayerNorm 的基本原理

LayerNorm 由 Ba, Kiros 和 Hinton 在 2016 年提出，首次应用于训练循环神经网络。

**核心思想**：对单个样本的所有特征进行归一化，不依赖 batch 维度。

### 2.2 数学公式

对于输入 $x \in \mathbb{R}^{B \times S \times D}$（batch, sequence, dimension），LayerNorm 的计算：

$$
\mu^{(i)} = \frac{1}{D} \sum_{d=1}^{D} x_d^{(i)}
$$

$$
\sigma^{(i)} = \sqrt{\frac{1}{D} \sum_{d=1}^{D} \left(x_d^{(i)} - \mu^{(i)}\right)^2}
$$

$$
\text{LN}(x)^{(i)} = \gamma \cdot \frac{x^{(i)} - \mu^{(i)}}{\sqrt{\sigma^{(i)2} + \epsilon}} + \beta
$$

其中 $i$ 表示第 $i$ 个样本，$D$ 是特征维度。

### 2.3 LayerNorm 的特点

**优点**：
- ✅ 不依赖 batch size，适用于 batch 较小或变化的场景
- ✅ 天然适合 RNN 和 Transformer
- ✅ 推理时计算稳定

**缺点**：
- ❌ 计算均值和标准差开销较大
- ❌ 在某些场景下不如 BatchNorm 有效

## 3. RMSNorm 详解

### 3.1 RMSNorm 的提出背景

RMSNorm 由 Zhang 和 Sennrich 在 2019 年提出（论文：*Root Mean Square Layer Normalization*）。

**核心动机**：LayerNorm 计算完整均值和标准差的开销较大，能否简化？

### 3.2 核心观察

作者发现 LayerNorm 中的均值项对最终效果贡献较小，主要贡献来自**均方根（Root Mean Square）**。

### 3.3 数学公式

RMSNorm 的计算：

$$
\text{RMSNorm}(x) = \gamma \cdot \frac{x}{\text{RMS}(x)}
$$

其中：

$$
\text{RMS}(x) = \sqrt{\frac{1}{D} \sum_{d=1}^{D} x_d^2} = \sqrt{\mathbb{E}[x^2]}
$$

即只计算输入的**均方根**，完全省略了均值的计算！

### 3.4 对比示意图

```
LayerNorm 计算流程：
┌─────────────────────────────────────┐
│  输入 x = [x₁, x₂, x₃, ..., x_D]    │
└─────────────────┬───────────────────┘
                  ↓
┌─────────────────────────────────────┐
│ 计算均值: μ = (1/D)Σxᵢ              │
│ 计算方差: σ² = (1/D)Σ(xᵢ-μ)²        │
│ 计算标准差: σ = √(σ² + ε)           │
└─────────────────┬───────────────────┘
                  ↓
┌─────────────────────────────────────┐
│  输出: γ · (x - μ)/σ + β           │
└─────────────────────────────────────┘

RMSNorm 计算流程：
┌─────────────────────────────────────┐
│  输入 x = [x₁, x₂, x₃, ..., x_D]    │
└─────────────────┬───────────────────┘
                  ↓
┌─────────────────────────────────────┐
│ 计算 RMS: RMS = √((1/D)Σxᵢ²)       │
│  （仅需一次遍历！）                  │
└─────────────────┬───────────────────┘
                  ↓
┌─────────────────────────────────────┐
│  输出: γ · x / RMS + β             │
└─────────────────────────────────────┘
```

## 4. 数学推导与对比

### 4.1 LayerNorm 完整推导

**步骤1：计算均值**

$$
\mu = \frac{1}{D}\sum_{d=1}^{D} x_d
$$

**步骤2：计算方差**

$$
\sigma^2 = \frac{1}{D}\sum_{d=1}^{D}(x_d - \mu)^2
$$

**步骤3：归一化**

$$
\hat{x}_d = \frac{x_d - \mu}{\sqrt{\sigma^2 + \epsilon}}
$$

**步骤4：仿射变换**

$$
y_d = \gamma_d \cdot \hat{x}_d + \beta_d
$$

### 4.2 RMSNorm 推导

**步骤1：计算均方根**

$$
\text{RMS}(x) = \sqrt{\frac{1}{D}\sum_{d=1}^{D} x_d^2}
$$

**步骤2：归一化**（直接除以 RMS）

$$
\hat{x}_d = \frac{x_d}{\text{RMS}(x)}
$$

**步骤3：仿射变换**

$$
y_d = \gamma_d \cdot \hat{x}_d + \beta_d
$$

### 4.3 关键差异对比

| 特性 | LayerNorm | RMSNorm |
|------|-----------|---------|
| **计算项** | 均值 + 方差 | 仅均方根 |
| **时间复杂度** | $O(2D)$ | $O(D)$ |
| **空间复杂度** | 需要保存 $\mu, \sigma^2$ | 仅需保存 RMS |
| **计算开销** | 较高 | 约节省 50% |
| **理论严谨性** | 零均值归一化 | 幅值归一化 |
| **梯度计算** | 复杂 | 相对简单 |

### 4.4 为什么 RMSNorm 能 work？

**直觉解释**：

1. **均值的平移不变性**：对于 reLU 等激活函数，$\text{ReLU}(x) = \max(0, x)$ 对正数部分不关心均值
2. **Attention 的特性**：在 Transformer 中，attention score 由 $QK^T$ 计算，加减常数不影响相对大小
3. **幅值归一化的有效性**：控制激活值的 scale 对训练稳定性更重要

**数学解释**：

设 $x = \mu + \sigma \cdot \hat{x}$，则：

$$
\text{LayerNorm}(x) = \gamma \cdot \hat{x} + \beta
$$

而 RMSNorm 的输入可以写成：

$$
x = \sigma \cdot \hat{x}'
$$

其中 $\hat{x}'$ 是单位 RMS 的向量（$\|\hat{x}'\|_2 = \sqrt{D}$）。

因此两种归一化只是在归一化方向上有差异，本质上都在做**单位化**。

### 4.5 形式化对比

$$
\text{LayerNorm}(x) = \gamma \cdot \frac{x - \text{mean}(x)}{\text{std}(x)} + \beta
$$

$$
\text{RMSNorm}(x) = \gamma \cdot \frac{x}{\text{RMS}(x)} + \beta
$$

展开来看，RMSNorm 相当于假设 $\text{mean}(x) = 0$。

## 5. 代码实现

### 5.1 PyTorch 实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class LayerNorm(nn.Module):
    """
    LayerNorm 实现
    支持对特定维度进行归一化
    
    Args:
        normalized_shape: 需要归一化的维度
        eps: 防止除零的小常数
        elementwise_affine: 是否使用可学习参数 γ, β
    """
    def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
        super().__init__()
        self.normalized_shape = (normalized_shape,) if isinstance(normalized_shape, int) else tuple(normalized_shape)
        self.eps = eps
        self.elementwise_affine = elementwise_affine
        
        # 可学习参数
        if self.elementwise_affine:
            self.weight = nn.Parameter(torch.ones(normalized_shape))
            self.bias = nn.Parameter(torch.zeros(normalized_shape))
        else:
            self.register_parameter('weight', None)
            self.register_parameter('bias', None)
    
    def forward(self, x):
        """
        前向传播
        
        Args:
            x: 输入张量，形状为 (batch, ..., normalized_shape)
        
        Returns:
            归一化后的张量
        """
        # 保存原始形状以便恢复
        original_shape = x.shape
        
        # 统一处理：把 normalized_shape 维度展平到最后
        # 计算需要保留的维度数量
        ndims = len(self.normalized_shape)
        x_reshaped = x
        
        if ndims > 1:
            # 多维度归一化：将中间维度展平
            # 例如: (B, S, D) -> (B*S, D) 或 (B, S, D) -> (B, S*D)
            if len(original_shape) > ndims:
                # 保留 batch 维度，展平中间和归一化维度
                x_reshaped = x.reshape(-1, self.normalized_shape[0])
        
        # 计算均值：沿归一化维度
        # keepdim=True 保持维度以便广播
        mean = x_reshaped.mean(dim=-1, keepdim=True)
        
        # 计算方差
        var = x_reshaped.var(dim=-1, keepdim=True, unbiased=False)
        
        # 归一化
        x_normalized = (x_reshaped - mean) / torch.sqrt(var + self.eps)
        
        # 应用仿射变换
        if self.elementwise_affine:
            x_normalized = x_normalized * self.weight + self.bias
        
        # 恢复原始形状
        return x_normalized.reshape(original_shape)


class RMSNorm(nn.Module):
    """
    RMSNorm (Root Mean Square Layer Normalization) 实现
    
    论文: Root Mean Square Layer Normalization (NeurIPS 2019)
    链接: https://arxiv.org/abs/1910.07467
    
    核心思想: 只计算均方根，省略均值的计算
    
    Args:
        normalized_shape: 需要归一化的维度
        eps: 防止除零的小常数
        elementwise_affine: 是否使用可学习参数 γ, β
    """
    def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
        super().__init__()
        self.normalized_shape = (normalized_shape,) if isinstance(normalized_shape, int) else tuple(normalized_shape)
        self.eps = eps
        self.elementwise_affine = elementwise_affine
        
        # 可学习参数（注意：RMSNorm 没有 bias）
        if self.elementwise_affine:
            self.weight = nn.Parameter(torch.ones(normalized_shape))
            # 注意：RMSNorm 默认没有 bias 项，这是与 LayerNorm 的区别
        else:
            self.register_parameter('weight', None)
    
    def forward(self, x):
        """
        前向传播
        
        Args:
            x: 输入张量，形状为 (batch, seq_len, hidden_dim) 或 (batch, hidden_dim)
        
        Returns:
            归一化后的张量
        
        数学原理:
            RMS(x) = sqrt(1/D * Σx_i²) = sqrt(E[x²])
            RMSNorm(x) = γ * x / RMS(x) + β
        """
        # 计算均方根 (Root Mean Square)
        # 注意：RMSNorm 只计算 RMS，不计算均值！
        
        # 保存原始形状
        original_shape = x.shape
        
        # 处理多维度归一化
        ndims = len(self.normalized_shape)
        x_reshaped = x
        
        if ndims > 1:
            if len(original_shape) > ndims:
                x_reshaped = x.reshape(-1, self.normalized_shape[0])
        
        # 计算均方根
        # RMS = sqrt(mean(x²))
        rms = torch.sqrt(
            x_reshaped.pow(2).mean(dim=-1, keepdim=True) + self.eps
        )
        
        # 归一化：直接除以 RMS（没有减去均值！）
        x_normalized = x_reshaped / rms
        
        # 应用仿射变换（只有 weight，没有 bias）
        if self.elementwise_affine:
            x_normalized = x_normalized * self.weight
        
        # 恢复原始形状
        return x_normalized.reshape(original_shape)


class RMSNormAttention(nn.Module):
    """
    RMSNorm 在 Attention 中的应用
    
    在 Transformer 中，RMSNorm 可以替换 LayerNorm，
    放在以下位置：
    1. Attention 之前 (pre-norm)
    2. FeedForward 之前 (pre-norm)
    3. 输出残差连接之后 (post-norm，较少使用)
    """
    def __init__(self, d_model, eps=1e-6):
        super().__init__()
        # RMSNorm 用于归一化 Q, K, V
        self.norm = RMSNorm(d_model, eps=eps)
    
    def forward(self, qkv):
        """
        对 Q, K, V 进行归一化
        
        Args:
            qkv: 形状为 (batch, seq_len, 3 * d_model) 或 (batch, seq_len, d_model)
        
        Returns:
            归一化后的 qkv
        """
        return self.norm(qkv)
```
