## Step 1: 理解层归一化（Layer Normalization）基础原理

**操作**: 了解层归一化的核心概念和计算公式

**代码**:
```python
import torch
import torch.nn as nn
import math

# LayerNorm 公式: 
# LayerNorm(x) = γ * (x - μ) / √(σ² + ε) + β
#
# 其中:
# μ = (1/H) * Σx_i    (均值)
# σ² = (1/H) * Σ(x_i - μ)²  (方差)
# γ, β: 可学习的缩放和偏移参数
# ε: 防止除零的小常数

class LayerNorm(nn.Module):
    def __init__(self, normalized_shape, eps=1e-5):
        super().__init__()
        self.gamma = nn.Parameter(torch.ones(normalized_shape))
        self.beta = nn.Parameter(torch.zeros(normalized_shape))
        self.eps = eps
    
    def forward(self, x):
        # x shape: (batch, seq_len, hidden_dim)
        mean = x.mean(dim=-1, keepdim=True)      # 计算均值
        var = x.var(dim=-1, keepdim=True, unbiased=False)  # 计算方差
        x_norm = (x - mean) / torch.sqrt(var + self.eps)   # 标准化
        return self.gamma * x_norm + self.beta

# 测试
batch, seq_len, hidden_dim = 2, 10, 512

output = layer_norm(x)

print(f"输入形状: {x.shape}")
print(f"输出形状: {output.shape}")
print(f"输出均值: {output.mean(dim=-1).mean().item():.6f}")
print(f"输出标准差: {output.std(dim=-1).mean().item():.6f}")
```

## Step 2: 实现RMSNorm（均方根归一化）

**操作**: 理解RMSNorm的核心思想——只使用RMS而非均值方差

**代码**:
```python
class RMSNorm(nn.Module):
    def __init__(self, normalized_shape, eps=1e-5):
        super().__init__()
        self.gamma = nn.Parameter(torch.ones(normalized_shape))
        self.eps = eps
    
    def forward(self, x):
        # RMSNorm 公式:
        # RMSNorm(x) = γ * x / RMS(x)
        # 其中 RMS(x) = √((1/H) * Σx_i²)
        
        rms = torch.sqrt(x.pow(2).mean(dim=-1, keepdim=True) + self.eps)
        x_norm = x / rms
        return self.gamma * x_norm

# 测试

print(f"RMSNorm 输出形状: {output_rms.shape}")
print(f"RMSNorm 输出均值: {output_rms.mean(dim=-1).mean().item():.6f}")
print(f"RMSNorm 输出RMS: {output_rms.pow(2).mean(dim=-1).mean().item():.6f}")
```

## Step 4: 数学推导——RMSNorm的数学原理

**操作**: 推导RMSNorm相对于LayerNorm的数学简化

**代码**:
```python
def mathematical_derivation():
    """
    RMSNorm 数学推导
    ================
    
    LayerNorm:
    --------
    1. 计算均值: μ = (1/H)Σx_i
    2. 中心化: x' = x - μ
    3. 计算方差: σ² = (1/H)Σ(x'_i)²
    4. 归一化: x'' = x' / √(σ² + ε)
    5. 仿射变换: y = γx'' + β
    
    步骤数: 5步 (需要计算均值和方差两次遍历)
    
    RMSNorm:
    --------
    1. 计算RMS: RMS = √((1/H)Σx_i²)
    2. 归一化: x' = x / RMS
    3. 仿射变换: y = γx' + β
    
    步骤数: 3步 (只需计算RMS一次遍历)
    
    关键洞察: 
    - RMSNorm 移除了均值中心化步骤
    - 当输入分布接近对称时，两者效果相近
    - 当输入分布有明显偏置时，LayerNorm会重新中心化
    """
    
    print("数学推导验证:")
    print("-" * 40)
    
    x = torch.tensor([3.0, 4.0])  # 向量示例
    
    # LayerNorm 过程
    mean_x = x.mean()
    centered = x - mean_x
    rms_centered = torch.sqrt((centered ** 2).mean())
    ln_result = centered / rms_centered
    
    print(f"原始向量: x = {x.tolist()}")
    print(f"\nLayerNorm:")
    print(f"  均值 μ = {mean_x.item()}")
    print(f"  中心化后: {centered.tolist()}")
    print(f"  RMS(中心化后) = {rms_centered.item()}")
    print(f"  LayerNorm结果: {ln_result.tolist()}")
    print(f"  结果均值 = {ln_result.mean().item()}")
    
    # RMSNorm 过程
    rms_x = torch.sqrt((x ** 2).mean())
    rn_result = x / rms_x
    
    print(f"\nRMSNorm:")
    print(f"  RMS(x) = {rms_x.item()}")
    print(f"  RMSNorm结果: {rn_result.tolist()}")
    print(f"  结果RMS = {torch.sqrt((rn_result ** 2).mean()).item()}")
    
    print(f"\n关系: LayerNorm(centered) = RMSNorm(x) * sign(x)")
    
mathematical_derivation()
```

```

----------------------------------------

  均值 μ = 3.5
  中心化后: [-0.5, 0.5]
  RMS(中心化后) = 0.5
  LayerNorm结果: [-1.0, 1.0]
```

## Step 5: 在MiniMind中应用RMSNorm

**操作**: 在Transformer架构中集成RMSNorm

```python
class RMSNormTransformerBlock(nn.Module):
    """
    MiniMind中使用的RMSNorm块
    替换标准LayerNorm以提升训练效率
    """
    def __init__(self, hidden_dim, num_heads, mlp_ratio=4):
        super().__init__()
        self.hidden_dim = hidden_dim
        
        # Pre-norm 结构 (Transformer标准)
        self.norm1 = RMSNorm(hidden_dim)
        self.attn = nn.MultiheadAttention(hidden_dim, num_heads, batch_first=True)
        
        self.norm2 = RMSNorm(hidden_dim)
        mlp_hidden = int(hidden_dim * mlp_ratio)
        self.mlp = nn.Sequential(
            nn.Linear(hidden_dim, mlp_hidden),
            nn.GELU(),
            nn.Linear(mlp_hidden, hidden_dim)
        )
    
    def forward(self, x):
        # Pre-norm + 残差连接
        x = x + self.attn(self.norm1(x), self.norm1(x), self.norm1(x))[0]
        x = x + self.mlp(self.norm2(x))
        return x

# 验证
print("RMSNorm Transformer块测试:")
block = RMSNormTransformerBlock(hidden_dim=256, num_heads=4)
x = torch.randn(2, 10, 256)  # batch=2, seq_len=10, hidden=256
```

## 核心公式速查表

| 特性 | LayerNorm | RMSNorm |
|------|-----------|---------|
| **公式** | `(x - μ) / √(σ² + ε)` | `x / √(x²的均值 + ε)` |
| **均值** | 归一化为0 | 不归一化 |
| **方差/RMS** | 归一化为1 | 归一化为1 |
| **计算步骤** | 5步 | 3步 |
| **参数** | γ, β | γ |
| **首次提出** | 2016 | 2023 |
