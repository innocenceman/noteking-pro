## 第一部分：大语言模型基础概念

### 1.1 什么是大语言模型 (LLM)

{KNOWLEDGE}背景知识{/KNOWLEDGE}

大语言模型（Large Language Model, LLM）是一类基于深度学习技术训练的文本处理模型，能够理解和生成人类语言文字。

```
传统方法: 规则系统 → 统计学习 → 机器学习
现代方法: 深度学习 → 预训练模型 → 大语言模型
```

LLM的核心特征：

| 特征 | 说明 |
|------|------|
| **规模庞大** | 参数量从数十亿到数千亿不等 |
| **通用能力** | 可处理多种NLP任务 |
| **涌现能力** | 参数量突破临界点后出现新能力 |
| **涌现能力** | 通过大规模文本自监督学习 |

### 1.2 Transformer 架构详解

{IMPORTANT}核心概念{/IMPORTANT}

Transformer是现代LLM的基石架构，由Google在2017年提出。它完全基于**注意力机制（Attention Mechanism）**，彻底摒弃了传统的RNN结构。

#### 1.2.1 整体架构

$$Transformer = Encoder + Decoder$$

{IMAGE:3}

**核心组件**：

```
┌─────────────────────────────────────────────┐
│              Transformer 架构               │
├─────────────────────────────────────────────┤
│  输入 Embedding → 位置编码 (Positional Encoding)  │
│              ↓                               │
│  ┌─────────────────────────────────────┐    │
│  │         多头自注意力层              │    │
│  │    (Multi-Head Self-Attention)      │    │
│  └─────────────────────────────────────┘    │
│              ↓                               │
│  ┌─────────────────────────────────────┐    │
│  │           前馈神经网络              │    │
│  │      (Feed Forward Network)         │    │
│  └─────────────────────────────────────┘    │
│              ↓                               │
│           Layer Norm → 残差连接             │
│              ↓                               │
│  输出 Projection                             │
└─────────────────────────────────────────────┘
```

#### 1.2.2 注意力机制公式

{IMAGE:4}

注意力机制的核心计算：

$$Attention(Q, K, V) = softmax\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

其中：
- $Q$ (Query): 查询向量，表示当前位置想要查找的信息
- $K$ (Key): 键向量，表示每个位置的索引信息
- $V$ (Value): 值向量，表示每个位置的实际内容
- $d_k$: 键向量的维度，用于缩放

多头的计算：

$$MultiHead(Q, K, V) = Concat(head_1, head_2, ..., head_h)W^O$$

$$head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)$$

#### 2.2.2 Transformer 层实现

{IMAGE:10}

```python
import math
import torch
import torch.nn as nn
import torch.nn.functional as F

class TransformerBlock(nn.Module):
    """Transformer编码器块"""
    
    def __init__(self, d_model, n_heads, d_ff, dropout=0.1):
        super().__init__()
        
        # 多头自注意力
        self.attention = nn.MultiheadAttention(
            embed_dim=d_model,
            num_heads=n_heads,
            dropout=dropout
        )
        
        # 前馈网络
        self.feed_forward = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.GELU(),
            nn.Linear(d_ff, d_model)
        )
        
        # 层归一化
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        # 自注意力 + 残差连接
        attn_output, _ = self.attention(x, x, x, attn_mask=mask)
        x = self.norm1(x + self.dropout(attn_output))
        
        # 前馈网络 + 残差连接
        ff_output = self.feed_forward(x)
        x = self.norm2(x + self.dropout(ff_output))
        
        return x
```

```python
class Tokenizer:
    """分词器基类"""
    
    def __init__(self, vocab_file):
        self.vocab = self._load_vocab(vocab_file)
        self.unk_token = "<unk>"
        self.pad_token = "<pad>"
    
    def encode(self, text):
        """将文本转换为token ids"""
        tokens = self._tokenize(text)
        return [self.vocab.get(t, self.vocab[self.unk_token]) 
                for t in tokens]
    
    def decode(self, ids):
        """将token ids转换回文本"""
        return ''.join([self.id_to_token[i] for i in ids])
```

## 第三部分：关键技术点深入

### 3.1 位置编码 (Positional Encoding)

$$PE_{(pos,2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

$$PE_{(pos,2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

```python
class PositionalEncoding(nn.Module):
    """正弦位置编码"""
    
    def __init__(self, d_model, max_len=5000):
        super().__init__()
        
        # 创建位置编码矩阵
        pe = torch.zeros(max_len, d_model)
        position = torch.arange(0, max_len).unsqueeze(1).float()
        
        # 计算除数项
        div_term = torch.exp(
            torch.arange(0, d_model, 2).float() * 
            (-math.log(10000.0) / d_model)
        )
        
        # 奇偶位置分别使用sin和cos
        pe[:, 0::2] = torch.sin(position * div_term)
        pe[:, 1::2] = torch.cos(position * div_term)
        
        self.register_buffer('pe', pe.unsqueeze(0))
    
    def forward(self, x):
        """添加位置编码"""
        return x + self.pe[:, :x.size(1)]
```

### 3.2 激活函数：GELU

{GELU (Gaussian Error Linear Unit)} 是现代Transformer模型中常用的激活函数，相比ReLU有更好的梯度流动特性：

$$GELU(x) = x \cdot \Phi(x) = x \cdot \frac{1}{2}[1 + \tanh(\sqrt{\frac{2}{\pi}}(x + 0.044715x^3))]$$

```python
class GELU(nn.Module):
    """高斯误差线性单元"""
    
    def __init__(self):
        super().__init__()
    
    def forward(self, x):
        # 近似实现
        return 0.5 * x * (1 + torch.tanh(
            math.sqrt(2 / math.pi) * 
            (x + 0.044715 * torch.pow(x, 3))
        ))
```
