## 1. Transformer 整体架构总览

{IMAGE:1}

Transformer 采用经典的 **Encoder-Decoder**（编码器-解码器）结构，整体呈对称设计。左侧为编码器，负责理解输入序列；右侧为解码器，负责生成输出序列。

{IMPORTANT}核心概念{/IMPORTANT}
- **Input**: 源序列（如中文句子）
- **Output**: 目标序列（如英文翻译）
- **N×**: 编码器和解码器各自堆叠 N 层（通常 N=6）

## 2. 输入嵌入层 (Input Embedding)

{IMAGE:2}

### 2.1 词嵌入向量

```python
import torch
import torch.nn as nn

class InputEmbedding(nn.Module):
    def __init__(self, vocab_size: int, d_model: int):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.d_model = d_model
    
    def forward(self, x):
        # x: [batch_size, seq_len] -> [batch_size, seq_len, d_model]
        return self.embedding(x) * math.sqrt(self.d_model)
```

{IMPORTANT}核心概念{/IMPORTANT}
- 将离散的词索引映射为连续的向量表示
- 典型维度：$d_{model} = 512$（小型）或 $d_{model} = 768$（大型）
- 乘以 $\sqrt{d_{model}}$ 是为了保持向量点积的方差稳定

## 3. 位置编码 (Positional Encoding)

{IMAGE:3}

位置编码是 Transformer 处理序列顺序的关键。由于 Self-Attention 本身不包含位置信息，必须显式注入。

### 3.1 三角函数位置编码公式

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

其中 $pos$ 是位置，$i$ 是维度索引。

### 3.2 位置编码实现

```python
import numpy as np

def get_positional_encoding(seq_len, d_model):
    """生成位置编码矩阵"""
    pe = np.zeros((seq_len, d_model))
    
    positions = np.arange(seq_len)[:, np.newaxis]
    div_term = np.exp(
        np.arange(0, d_model, 2) * (-np.log(10000.0) / d_model)
    )
    
    pe[:, 0::2] = np.sin(positions * div_term)
    pe[:, 1::2] = np.cos(positions * div_term)
    
    return torch.tensor(pe, dtype=torch.float32)
```

{IMAGE:4}

{WARNING}易错点{/WARNING}
- 位置编码直接加到词嵌入上，而非拼接
- 预训练模型（如 BERT）也支持可学习的位置编码，但效果与三角函数版本相当

## 4. 编码器层 (Encoder Layer)

{IMAGE:5}

每个编码器层包含两个子层：
1. **Multi-Head Self-Attention**（多头自注意力）
2. **Feed Forward Network**（前馈神经网络）

每个子层周围都有 **残差连接** 和 **Layer Normalization**。

## 5. 多头自注意力机制 (Multi-Head Attention)

{IMAGE:6}

### 5.1 Scaled Dot-Product Attention

{IMAGE:7}

这是注意力机制的核心计算单元：

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

```python
import torch
import torch.nn.functional as F
import math

def scaled_dot_product_attention(Q, K, V, mask=None):
    """
    Q, K, V: [batch, heads, seq_len, d_k]
    返回: [batch, heads, seq_len, d_v]
    """
    d_k = Q.size(-1)
    
    # 计算注意力分数
    scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(d_k)
    
    # 应用掩码（可选）
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)
    
    # softmax归一化
    attention_weights = F.softmax(scores, dim=-1)
    
    # 加权求和
    output = torch.matmul(attention_weights, V)
    
    return output, attention_weights
```

{IMPORTANT}核心概念{/IMPORTANT}
- $\sqrt{d_k}$ 的缩放因子防止点积过大导致 softmax 梯度消失
- 掩码机制用于处理变长序列和防止信息泄露

### 5.2 多头机制

{IMAGE:8}

```python
class MultiHeadAttention(nn.Module):
    def __init__(self, d_model, num_heads):
        super().__init__()
        assert d_model % num_heads == 0
        
        self.d_model = d_model
        self.num_heads = num_heads
        self.d_k = d_model // num_heads
        
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
    
    def forward(self, Q, K, V, mask=None):
        batch_size = Q.size(0)
        
        # 线性变换后分头
        Q = self.W_q(Q).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        K = self.W_k(K).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        V = self.W_v(V).view(batch_size, -1, self.num_heads, self.d_k).transpose(1, 2)
        
        # 计算注意力
        attn_output, _ = scaled_dot_product_attention(Q, K, V, mask)
        
        # 合并多头
        attn_output = attn_output.transpose(1, 2).contiguous().view(
            batch_size, -1, self.d_model
        )
        
        return self.W_o(attn_output)
```

{IMAGE:9}

{KNOWLEDGE}背景知识{/KNOWLEDGE}
- 多头注意力让模型能够同时关注不同子空间的信息
- 典型配置：8个头，$d_k = d_v = d_{model}/8 = 64$
- 每个头学习不同的注意力模式（如语法、语义、位置等）

## 6. 前馈神经网络 (Feed Forward Network)

{IMAGE:10}

```python
class FeedForward(nn.Module):
    def __init__(self, d_model, d_ff=2048, dropout=0.1):
        super().__init__()
        self.linear1 = nn.Linear(d_model, d_ff)
        self.linear2 = nn.Linear(d_ff, d_model)
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x):
        return self.linear2(self.dropout(F.gelu(self.linear1(x))))
```

{IMPORTANT}核心概念{/IMPORTANT}
- 两层线性变换，中间层维度扩展（通常 $d_{ff} = 4 \times d_{model}$）
- GELU 激活函数已被证明优于 ReLU
- 这是一个逐位置（position-wise）的变换，独立作用于每个 token

## 7. 层归一化与残差连接

{IMAGE:11}

{IMPORTANT}核心概念{/IMPORTANT}
Layer Normalization 公式：

$$\text{LayerNorm}(x) = \gamma \cdot \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$$

```python
class EncoderLayer(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        self.attention = MultiHeadAttention(d_model, num_heads)
        self.norm1 = nn.LayerNorm(d_model)
        self.ffn = FeedForward(d_model, d_ff, dropout)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        # 自注意力 + 残差连接
        attn_output = self.attention(x, x, x, mask)
        x = self.norm1(x + self.dropout(attn_output))
        
        # FFN + 残差连接
        ffn_output = self.ffn(x)
        x = self.norm2(x + self.dropout(ffn_output))
        
        return x
```

{IMAGE:12}

{WARNING}易错点{/WARNING}
- 残差连接需要维度匹配，否则无法直接相加
- LayerNorm 放在残差路径内部（Post-LN）而非外部（Pre-LN）

## 8. 编码器堆叠

完整的编码器由 N 个 EncoderLayer 堆叠而成：

```python
class Encoder(nn.Module):
    def __init__(self, num_layers, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        self.layers = nn.ModuleList([
            EncoderLayer(d_model, num_heads, d_ff, dropout)
            for _ in range(num_layers)
        ])
        self.norm = nn.LayerNorm(d_model)
    
    def forward(self, x, mask=None):
        for layer in self.layers:
            x = layer(x, mask)
        return self.norm(x)
```

## 9. 解码器结构 (Decoder)

解码器与编码器类似，但增加了额外的 **Cross Attention** 层：

1. **Masked Self-Attention**（掩码自注意力）- 防止看到未来位置
2. **Cross Attention**（交叉注意力）- 使用编码器输出作为 K、V
3. **Feed Forward Network**

```python
class DecoderLayer(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        self.masked_attention = MultiHeadAttention(d_model, num_heads)
        self.norm1 = nn.LayerNorm(d_model)
        
        self.cross_attention = MultiHeadAttention(d_model, num_heads)
        self.norm2 = nn.LayerNorm(d_model)
        
        self.ffn = FeedForward(d_model, d_ff, dropout)
        self.norm3 = nn.LayerNorm(d_model)
    
    def forward(self, x, encoder_output, src_mask=None, tgt_mask=None):
        # Masked Self-Attention
        mask_attn = self.masked_attention(x, x, x, tgt_mask)
        x = self.norm1(x + mask_attn)
        
        # Cross Attention
        cross_attn = self.cross_attention(x, encoder_output, encoder_output, src_mask)
        x = self.norm2(x + cross_attn)
        
        # FFN
        x = self.norm3(x + self.ffn(x))
        
        return x
```

## 10. 输出层

解码器输出经过线性变换和 Softmax，生成下一个 token 的概率分布：

```python
class Transformer(nn.Module):
    def __init__(self, vocab_size, d_model, num_layers, num_heads, d_ff):
        super().__init__()
        self.encoder = Encoder(num_layers, d_model, num_heads, d_ff)
        self.decoder = Decoder(num_layers, d_model, num_heads, d_ff)
        self.output_proj = nn.Linear(d_model, vocab_size)
    
    def forward(self, src, tgt):
        # 编码源序列
        encoder_output = self.encoder(src)
        
        # 解码目标序列
        decoder_output = self.decoder(tgt, encoder_output)
        
        # 生成概率分布
        logits = self.output_proj(decoder_output)
        
        return F.log_softmax(logits, dim=-1)
```

| 组件 | 核心作用 |
