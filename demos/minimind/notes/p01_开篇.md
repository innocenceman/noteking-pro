## 数学基础回顾

### 神经网络核心数学

本课程涉及以下数学知识，建议提前复习：

#### 1. 矩阵运算

$$Y = X \cdot W + b$$

其中 $X$ 为输入矩阵，$W$ 为权重矩阵，$b$ 为偏置向量。

#### 2. Softmax 函数

$$\text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_{j} e^{x_j}}$$

用于将任意实数向量转换为概率分布。

#### 3. 交叉熵损失

$$\mathcal{L} = -\sum_{i} y_i \cdot \log(\hat{y}_i)$$

衡量预测分布与真实分布之间的差异，是分类任务常用的损失函数。

#### 4. 链式法则

$$\frac{\partial \mathcal{L}}{\partial \theta} = \frac{\partial \mathcal{L}}{\partial \mathcal{f}} \cdot \frac{\partial \mathcal{f}}{\partial \theta}$$

反向传播算法的数学基础。

{KNOWLEDGE}背景知识{/KNOWLEDGE}

> **PyTorch 如何自动求导？**
> 
> PyTorch 通过 `autograd` 模块自动构建计算图。当你执行前向传播时，PyTorch 会记录所有操作；调用 `.backward()` 时，它会根据链式法则自动计算所有参数的梯度。

## PyTorch 快速入门

### 张量基础

张量（Tensor）是 PyTorch 中的核心数据结构，可以理解为多维数组。

```python
import torch

# 创建张量的几种方式
# 1. 从 Python 列表创建
x = torch.tensor([[1, 2], [3, 4]], dtype=torch.float32)

# 2. 随机初始化
w = torch.randn(512, 768)  # 正态分布随机
b = torch.zeros(512)       # 全零初始化

# 3. 从 numpy 数组转换
import numpy as np
np_array = np.array([[1, 2], [3, 4]])
x = torch.from_numpy(np_array).float()

# 张量基本操作
print(f"形状: {x.shape}")
print(f"数据类型: {x.dtype}")
print(f"设备: {x.device}")
```

### 自动求梯度

```python
# 开启梯度追踪
x = torch.tensor([1.0, 2.0, 3.0], requires_grad=True)

# 定义计算图
y = x ** 2
z = y.mean()

# 反向传播，计算梯度
z.backward()

# 查看梯度
print(x.grad)  # 输出: tensor([0.6667, 1.3333, 2.0000])
```

{IMPORTANT}核心概念{/IMPORTANT}

> **计算图（Computational Graph）**
> 
> PyTorch 使用动态计算图（Dynamic Computational Graph）。每次前向传播时，图会被实时构建，这使得调试更加直观，也方便使用 Python 的控制流语句。

### 代码实践要点

```python
# 学习代码时的正确姿势

# ❌ 不要只是复制粘贴
model = Transformer()

# ✅ 要理解每一行代码的作用
class Transformer(nn.Module):
    def __init__(self, vocab_size, d_model, nhead, num_layers):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, d_model)
        self.transformer = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(
                d_model=d_model, 
                nhead=nhead
            ),
            num_layers=num_layers
        )
        self.fc = nn.Linear(d_model, vocab_size)
    
    def forward(self, x):
        x = self.embedding(x)
        x = self.transformer(x)
        x = self.fc(x)
        return x
```

>

> output = model(input)  # input 还在 CPU 上！
> 
> # 正确做法
> device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
> model = model.to(device)
> input = input.to(device)
> output = model(input)
> ```
