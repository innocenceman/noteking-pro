### 1.1 数据处理模块的修正

{IMAGE:2}

在Ep1的数据处理代码中，存在一个**索引越界**的潜在风险。当数据集的最后一个batch不足以填满设定的batch_size时，代码会抛出异常。

{IMAGE:3}

**修正后的代码示例：**

```python
# 原始代码（存在问题）
for i in range(0, len(dataset), batch_size):
    batch = dataset[i:i + batch_size]  # 当 i + batch_size > len(dataset) 时会出错
    # 处理batch...

# 修正后的代码
for i in range(0, len(dataset), batch_size):
    end_idx = min(i + batch_size, len(dataset))
    batch = dataset[i:end_idx]  # 安全地处理最后一个不完整的batch
    # 处理batch...
```

### 1.2 模型初始化参数修正

{IMAGE:5}

在多头注意力机制的参数初始化部分，原始代码使用了不恰当的初始化方法，可能导致训练初期梯度不稳定。

```python
# 修正后的参数初始化
class MultiHeadAttention(nn.Module):
    def __init__(self, d_model, num_heads):
        super().__init__()
        assert d_model % num_heads == 0
        
        self.d_model = d_model
        self.num_heads = num_heads
        self.d_k = d_model // num_heads
        
        # 使用 Xavier 均匀初始化替代之前的随机初始化
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
        
        # 添加参数初始化
        nn.init.xavier_uniform_(self.W_q.weight)
        nn.init.xavier_uniform_(self.W_k.weight)
        nn.init.xavier_uniform_(self.W_v.weight)
        nn.init.xavier_uniform_(self.W_o.weight)
```

{IMAGE:6}

{KNOWLEDGE}**为什么初始化很重要？**
深度学习中的参数初始化对训练稳定性有决定性影响。Xavier初始化（也称为Glorot初始化）基于以下考虑：
- 确保前向传播和反向传播时方差保持一致
- 避免梯度消失或梯度爆炸
- 使网络各层学习速度相对均衡
{/KNOWLEDGE}

## 第二部分：理论公式补充

### 2.1 注意力机制的数学推导

{IMAGE:7}

本节补充注意力机制的完整数学公式，这是理解Transformer架构的核心。

**注意力分数计算公式：**

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

其中：
- $Q \in \mathbb{R}^{n \times d_k}$：查询矩阵（Query）
- $K \in \mathbb{R}^{m \times d_k}$：键矩阵（Key）
- $V \in \mathbb{R}^{m \times d_v}$：值矩阵（Value）
- $d_k$：键向量的维度

{IMAGE:8}

**缩放因子的意义：**

{IMPORTANT}为什么要除以 $\sqrt{d_k}$？{/IMPORTANT}

当 $d_k$ 较大时，$QK^T$ 的点积结果会方差变大，导致softmax函数进入饱和区域（梯度接近于零）。除以 $\sqrt{d_k}$ 可以将方差归一化，保持梯度的稳定传递。

数学推导：
$$\text{Var}(q \cdot k) = \text{Var}\left(\sum_{i=1}^{d_k} q_i k_i\right) = d_k \cdot \text{Var}(q_i) \cdot \text{Var}(k_i)$$

假设 $q_i$ 和 $k_i$ 独立同分布且均值为0，则：
$$\text{Var}(q \cdot k) = d_k \cdot \sigma^4$$

除以 $\sqrt{d_k}$ 后，方差变为 $\sigma^2$，与维度无关。

### 2.2 多头注意力的表达公式

{IMAGE:9}

$$MultiHead(Q, K, V) = \text{Concat}(head_1, head_2, ..., head_h)W^O$$

其中每个头的计算：
$$head_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

{IMAGE:10}

**参数矩阵维度：**
- $W_i^Q, W_i^K, W_i^V \in \mathbb{R}^{d_{model} \times d_k}$
- $W^O \in \mathbb{R}^{hd_v \times d_{model}}$

## 第三部分：关键概念强化

### 3.1 位置编码的重要性

{IMAGE:11}

Transformer架构本身是**置换不变**的——对输入序列的任意排列，输出都相同。这与自然语言的语序敏感性形成矛盾。位置编码（Positional Encoding）是解决这一问题的标准方法。

{WARNING}常见误解：很多学习者认为位置编码只是简单地给每个位置分配一个数字（1, 2, 3, ...）。实际上，原始Transformer使用的是基于正弦和余弦函数的位置编码，这种设计允许模型学习相对位置关系。{/WARNING}

**位置编码公式：**

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

其中：
- $pos$：位置索引（0, 1, 2, ...）
- $i$：维度索引
- $d_{model}$：模型维度

{IMAGE:12}

### 3.2 残差连接与层归一化

残差连接（Skip Connection）是Transformer训练成功的关键因素之一。其数学形式为：

$$\mathbf{y} = \text{LayerNorm}(\mathbf{x} + \text{Sublayer}(\mathbf{x}))$$

{KNOWLEDGE}**残差连接的直观理解：**
残差连接创建了一条"捷径"，使得梯度可以无损地传播到浅层网络。这解决了深层网络训练困难的问题，使得构建数十层甚至数百层的Transformer成为可能。{/KNOWLEDGE}

## 第四部分：代码实现细节补充

### 4.1 完整的Transformer编码器层实现

```python
import torch
import torch.nn as nn
import math

class PositionalEncoding(nn.Module):
    """位置编码实现"""
    def __init__(self, d_model, max_len=5000):
        super().__init__()
        
        # 创建位置编码矩阵
        pe = torch.zeros(max_len, d_model)
        position = torch.arange(0, max_len, dtype=torch.float).unsqueeze(1)
        div_term = torch.exp(
            torch.arange(0, d_model, 2).float() * (-math.log(10000.0) / d_model)
        )
        
        pe[:, 0::2] = torch.sin(position * div_term)  # 偶数维度
        pe[:, 1::2] = torch.cos(position * div_term)  # 奇数维度
        pe = pe.unsqueeze(0)  # 添加batch维度
        
        # 注册为buffer（不参与梯度计算，但会随模型保存）
        self.register_buffer('pe', pe)
    
    def forward(self, x):
        """将位置编码添加到输入"""
        return x + self.pe[:, :x.size(1), :]


class TransformerEncoderLayer(nn.Module):
    """Transformer编码器层（修正版）"""
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        
        # 多头自注意力
        self.self_attn = nn.MultiheadAttention(
            d_model, num_heads, dropout=dropout, batch_first=True
        )
        
        # 前馈神经网络
        self.linear1 = nn.Linear(d_model, d_ff)
        self.linear2 = nn.Linear(d_ff, d_model)
        
        # 层归一化
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        
        # Dropout
        self.dropout = nn.Dropout(dropout)
        
        # FFN激活函数
        self.activation = nn.GELU()
    
    def forward(self, x, mask=None):
        # 自注意力 + 残差连接
        attn_output, _ = self.self_attn(x, x, x, attn_mask=mask)
        x = self.norm1(x + self.dropout(attn_output))
        
        # 前馈网络 + 残差连接
        ff_output = self.linear2(self.activation(self.linear1(x)))
        x = self.norm2(x + self.dropout(ff_output))
        
        return x
```
